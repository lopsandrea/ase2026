# PRATICA-DAI

> E-commerce checkout sandbox with async cart and session-based authentication.
> Used as Doc2Test evaluation subject (paper §4.1, "PRATICA-DAI" row).

- **Stack:** Node.js 20 + Express + EJS + express-session.
- **Workflow archetype:** transactional checkout (`e-Commerce` row in Tab. 1).
- **Views:** ~20 (catalog, product, cart, checkout, payment-mock, orders, …).
- **License:** Apache-2.0.

## Run

```bash
npm install
npm start          # http://localhost:3000
```
