# Taboo

> Real-time WebSocket board game with custom JS event listeners and a 60s timer per turn.
> Doc2Test evaluation subject (paper §4.1, "Taboo" row).

- **Stack:** Node.js 20 + ws + plain HTML/JS client.
- **Workflow archetype:** Real-time WebSocket flow (`Game Flow` row in Tab. 1).
- **License:** Apache-2.0.

```bash
npm install && npm start    # http://localhost:3000
```
